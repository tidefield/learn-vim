# LearnVim

A macOS menu bar app to learn Vim with a local AI assistant.

## Features

- **AI Chat** — Ask questions in plain English, get keystrokes back. Powered by Llama 3.2 3B running locally on your Mac.
- **Cheat Sheet** — Searchable quick reference of common Vim commands
- **Global Hotkey** — Press ⌥V to toggle from anywhere

## Requirements

- macOS 14+
- Apple Silicon (M1/M2/M3/M4)
- ~2GB disk for the AI model (downloaded on first use)

## Install

1. Download `LearnVim.zip` from [Releases](https://github.com/tidefield/learn-vim/releases)
2. Unzip and move to Applications
3. Right-click → Open (first time only, to bypass Gatekeeper)

## Usage

- Click the Vim icon in your menu bar or press **⌥V**
- Type a question like "how do I delete inside quotes?"
- Get back: `di"`
